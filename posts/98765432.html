Cloudflare Pages 国内访问优化 -- 利用「优选 IP」给 Logo 库 / 直播源订阅加速

发表于 2026-01-13|更新于2026-01-13|技术
|字数总计:1229|阅读时长:4分钟|阅读量:

📺 前言

在上一篇文章里，我们已经用 Cloudflare Pages 搭好了自己的频道 Logo 图库 + 直播源托管。用下来会发现一个很现实的问题：稳定是稳定了，但国内访问速度有时并不理想。

这是 Cloudflare 的老问题了——节点在国外多，国内直连偶尔慢、偶尔抽风，尤其在播放器里加载台标时更明显。

那有没有办法不换平台、不搬服务器，只通过配置，就把访问速度优化一下？答案是：可以。

这篇就来分享一个我自己在用的方案：

👉 Cloudflare Pages + 华为云国际站 + 第三方 CNAME 优选 IP

🧠 整体思路

核心逻辑只有一句话：

  • 国外继续走 Cloudflare
  • 中国大陆单独走「优选 IP」线路

而实现这个效果，靠的是三点：

  1. Cloudflare 子域名 NS 下沉
  2. 华为云国际站 线路解析
  3. 第三方 Cloudflare 优选 IP / CNAME

最终效果是：

  • 国外访问：走 Cloudflare 官方线路
  • 国内访问：命中优选 IP，速度明显改善
  • Pages 项目 完全不用改

🗺️ 优选 IP 架构示意

可以简单理解成这样：

你只是在 DNS 层做了分流，Pages 项目本身完全感知不到变化。

📌 准备条件

在开始之前，你需要准备好：
✅ 一个 Cloudflare Pages 项目，例如上一篇我们搭建的 logo 库
✅ 一个 Pages 使用的 二级域名
✅ 一个 第三方 Cloudflare 优选 IP / CNAME
✅ 注册 华为云国际站 账号（无需实名，无需手机绑定）

  1. 假设你给 Pages 绑定的域名是：logo.example.com

☁️ 二、注册并使用华为云国际站

注意是 国际站，不是国内站。

注册地址

用邮箱即可注册，无需实名，也无需绑定海外手机。

注册完成后进入控制台 - 云解析服务 DNS。

🧩 三、在华为云添加子域名解析

1️⃣ 添加域名

在华为云 DNS 中，创建公网域名：

⚠️ 这里只添加二级域名，不是主域名。因为我的主域名托管在 cloudflare。

2️⃣ 添加两条 DNS 记录（关键步骤）

🔹 第一条：默认线路（非大陆）

  • 类型：CNAME
  • 主机记录：空
  • 线路类型：全网默认
  • 解析值：Cloudflare Pages 分配的 xxx.pages.dev

📌 这条保证：海外访问非大陆线路正常走 Cloudflare 官方网络。

🔹 第二条：中国大陆线路（优选 IP）

  • 类型：CNAME
  • 主机记录：空
  • 线路类型：地域解析/中国大陆
  • 解析值：第三方提供的 优选 IP CNAME
    例如：logo.cf.090227.xyz

📌 这条才是核心：大陆访问全部走优选 IP。

记录华为云分配的 NS 服务器域名

把这组 NS 地址记下来。

🔁 四、回到 Cloudflare 设置 NS 下沉

现在回到 Cloudflare 域名管理 → DNS。

添加 NS 记录：

  • 类型：NS
  • 名称：logo
  • 名称服务器：华为云分配的 NS
  • TTL：默认自动

📌 这一步的含义是：
logo.example.com 这个子域名 DNS 解析权下放给华为云，而主域名仍然完全由 Cloudflare 管理。

✅ 五、验证效果

配置完成后，一般几分钟到十几分钟生效。你可以通过以下方式验证：

  1. itdog 等网页 tcping 测试域名
  2. 浏览器直接访问 Logo
  3. 播放器里加载台标
  4. 使用不同网络测试（大陆 / 非大陆）

常见现象：

  • 国内访问速度明显改善
  • 台标加载更快
  • Pages 项目无需任何改动

对比未优选 IP 的情况，加速效果明显。

🎯 总结

通过 Cloudflare Pages + 华为云国际站 + 优选 IP，你可以在 不更换平台、不改项目 的前提下：

🚀 明显改善国内访问速度
🌍 保留 Cloudflare 全球 CDN
🔐 架构清晰、风险可控

对于 Logo 图库、直播源订阅、静态资源托管 这种场景来说，这是一个性价比非常高的优化方案。

⚠️ 注意事项 & 经验提醒

📌 优选 IP 质量 > 数量，别乱用
📌 优选 IP 挂了不影响海外线路
📌 子域 NS 下沉不会影响主域名
📌 不要在 Cloudflare 再额外加 CNAME
